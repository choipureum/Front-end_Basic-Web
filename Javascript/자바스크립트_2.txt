< JS 데이터 타입 >

 - Number : 숫자 	보라색

 - String : 문자	검은색

 - Boolean : 논리	파란색

 - Object : 객체	{객체}
			{키1:값1, 키2:값2...}

 - Array : 배열 	[배열요소1, 배열요소2,....]

 - NULL : null값(참조대상이 없음을 나타내는 키워드)

 - undefined : 정의되지 않은 상태를 표현하는 키워드

 - NaN : Not a Number
	Number타입의 특수한 값(키워드)

 	Number타입으로 값이 인식되어야하는 시점에 Number가 아닌 데이터를 사용한 경우 리턴되는
	특수한 값

 - Infinity : 무한대
	Number타입의 특수한 값(키워드)
	양의 무한대를 나타내는 키워드
	특정 숫자를 0으로 나누었을 때 표현되는 특수한 값
	
	-(마이너스) 부호를 붙여 음의 무한대도 나타낼 수 있다(-Infinity)
