< 자바스크립트, Javascript > 

 - HTML문서에서 동작을 제어하는 언어

 	ex) <button>을 클릭했을 때 발생하는 이벤트 처리(event listener)
	    <form>에 입력한 값이 적절한 지 판단하기(validation)

 - HTML문서의 <head> 태그에 <script> 태그를 작성하여 자바스크립트 코드를 작성하여 사용한다

 - 브라우저의 자바스크립트엔진에 의해 인터프리트방식으로 동작한다

 - 자바스크립트 파일(.js)을 HTML문서에서 로드해서 사용하기도 한다

 - 자바스크립트 코드(Front side Script)는 브라우저에서 동작한다.

 	-> 코드가 브라우저에게 전달되어야 실행된다

	-> 민감한 정보나 핵심코드를 입력해서는 안된다
